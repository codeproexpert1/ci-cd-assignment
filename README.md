# CI/CD Assignment - GitHub Actions Testing

This repository demonstrates automated testing using GitHub Actions for a Python script that finds the lowest number in a file.

## Project Structure
- `find_lowest_number.py` - Python script to find lowest number
- `test_data/` - Test cases with input and expected output files
- `.github/workflows/test.yml` - GitHub Actions workflow configuration

## Test Cases
1. File with 10 numbers
2. File with 1 number
3. Empty file
